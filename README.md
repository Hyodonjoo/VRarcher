# VRarcher 게임 개발 프로젝트
VR 환경에서 활을 쏴서 점수를 1점씩 가산해 가는 게임 개발

# 개발기간
- 2024.05 ~ 2024.06

## 담당
| 이름 | 담당 역할 및 기능 |
| ------ | ------ |
| 주효돈 | PM, 기획, 메인 UI 및 활과 활시위 매커니즘 구현 |
| 김동옥 | 점수 오브젝트 및 점수 계산 구현 |
| 김웅희 | 맵 및 움직이는 배경 오브젝트 구현|
